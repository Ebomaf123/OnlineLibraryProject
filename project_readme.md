Online Library Management System

This is a complete and functional library management system, developed using Python for the backend (Flask), business logic (Object-Oriented classes), and the client frontend (Tkinter).

Development Stages (Completed)

Setup of the static web presentation page.

Development of the core data management classes.

Implementation of the REST API server.

Creation of the client GUI for management.

Unit and integration testing.

Finalization of documentation and the README.

Getting Started (Mac/Python 3)

Clone the repository.

Install dependencies (Flask, Requests; Tkinter is built-in with Python 3) with pip3 install Flask requests

Launch the API server with python3 backend_server.py.

Launch the GUI client with python3 frontend_app.py.

The project is ready for initial deployment.
